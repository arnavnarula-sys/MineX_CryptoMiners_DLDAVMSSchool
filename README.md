# 🔐 MineX - Cryptocurrency Mining & Quantum Computing

An interactive educational platform designed to make cryptocurrency mining and quantum computing easy to understand through animations, visualizations, and hands-on demonstrations.

Built as a Science Exhibition project, this website explains the core technologies behind cryptocurrencies without requiring any prior knowledge.

---

## 📖 About

Cryptocurrency mining often introduces terms such as **hashing**, **SHA-1**, **blockchain**, **proof of work**, **nonces**, **difficulty**, and **quantum computing** that can be difficult for beginners.

This project simplifies these concepts using interactive visualizations instead of lengthy theoretical explanations.
