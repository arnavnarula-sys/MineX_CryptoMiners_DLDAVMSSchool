// MINING PROCESS

From Start to End

Four steps to join the Duino Coin network and start earning DUCO.

01

Mining Job

Duino Coin Server provides the mining job to the miner giving them input data based on their predictions.

02

Computing Your Hashes

Miner through the Arduino computes hashes using DUCO S1 calculating different hashes and verifying result simultaneously.

03

Verification

Once the user submits the hash, Server verifies it to see if it matches their expected hash.

04

Reward

If verified correctly, server rewards the miner with a DUCO. If it doesn't match, the miner keeps trying nonces until verification succeeds.
